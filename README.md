# GottaHackEmAll
You can't directly run this using this repo. I reccomend you to try it out using the hack club workspace - prefferably in the #hackemon channel and then simply use any / commands you like. If you wish to run it on a different channel dont forget to mention it first using - @GottaHackEm'All 

SLASH COMMANDS
/hackatime link <username> - do this before anything
/catch <number> - catches a pokemon - adding a negative number or 0 or 1 behind it gives you only a 70% chance to catch a pokemon but spends only 1 minute
                                    - adding a number greater than 1 gives you a 100% chance to catch a pokemon but spends 3 minutes
/queststatus - checks your proggress on your quests
/poketeam - shows you your team
/pokedex - shows you your pokedex proggress (How many pokemon you have caught from gen 1) (out of 151)
/pokestats - shows you your stats
/battle <@user> - battles a... user - but however 1. you cant battle yourself, so if your'e testing this alone that's a bummmer
                                                  2. even if you have a partner both of you need to link your hackatimes and also ensure you have atleast 1 minute so you can                                                       catch a pokemon and actually battle
/questclaim - claims xp from your quest
/levelup <number> - levels up the pokemon who is numbered at the number you put in 
                  -  if this sounds confusing just write /levelup to get a quick visual explenation as well as your list of pokemon so you can see which one is numbered at                         what
